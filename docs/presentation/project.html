Project Overview

The Webserv project is a custom HTTP server created as part of School 42's curriculum. Inspired by Nginx, it allows handling HTTP requests and responses, supporting various HTTP methods, and mimicking real-world server configurations.

Key Features

HTTP 1.1 Compliance

Fully compliant with the HTTP/1.1 protocol, enabling standard request and response handling.

Configurable Server

Customizable configuration files to set up server blocks, ports, root directories, and more.

Error Handling

Handles HTTP errors gracefully with custom error pages for enhanced user experience.

Multi-threaded Processing

Supports multi-threaded request handling to manage concurrent connections efficiently.
